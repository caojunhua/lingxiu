【主祷文】
我们在天上的父，
愿人都尊你的名为圣，
愿你的国降临，
愿你的旨意行在地上，
如同行在天上，
我们日用的饮食，
今日赐给我们，
免我们的债，
如同我们免了人的债，
不叫我们遇见试探，
救我们脱离凶恶，
因为国度、权柄、荣耀，
都是你的，
直到永远，
阿们！
